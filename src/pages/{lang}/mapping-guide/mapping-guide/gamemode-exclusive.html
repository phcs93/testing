On the original game, by giving a palete value of 1 (full blue) to weapons, ammo, items, inventory items, buttons, cracks and cameras, these sprites would then be exclusive to multiplayer matches, meaning they would't spawn at all if playing a single player game. No jogo original, ao colocar a paleta de valor 1 (azul) em armas, munições, itens, itens do inventário, botões, rachaduras e cameras, essas sprites então tornam-se exclusivas de partidas multiplayer, ou seja, elas nunca aparecerão em partidas single player. That idea was extended further on proDuke, so the mapper can set actual filters so certain sprites can only appear on some game modes. This improves the potential for map balance, so it's not needed to make different versions of the same map for different game modes. Essa ideia foi extendida no proDuke, para que o criador de mapas possa setar filtros para que as sprites apareçam somente em certos modos de jogo. Isso aumenta o potencial de balanceamento do mapa, de forma que não se faz necessário criar diferentes versões de um mapa para modos de jogo diferentes.
